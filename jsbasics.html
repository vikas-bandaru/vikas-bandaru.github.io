JavaScript Basics Tutorial

Welcome to our tutorial on the basics of JavaScript! In this tutorial, we will cover variables, data types, operators, conditional statements, arrays, objects, and loops, and provide examples and expected results.

Variables

Variables are used to store values. In JavaScript, we can declare variables using the var keyword. Here's an example:

var x = 5;

In this example, we've declared a variable x and assigned it the value of 5.

Data Types

JavaScript supports several data types:

  • Numbers - e.g. 42, 3.14
  • Strings - e.g. "hello", 'world'
  • Booleans - e.g. true, false
  • Arrays - e.g. [1, 2, 3], ['apple', 'banana', 'cherry']
  • Objects - e.g. {name: "John", age: 30}
  • Undefined - e.g. undefined
  • Null - e.g. null

Operators

JavaScript supports several types of operators:

  • Arithmetic operators - e.g. +, -, *, /, %
  • Assignment operators - e.g. =, +=, -=, *=, /=, %=
  • Comparison operators - e.g. ==, !=, ===, !==, <, >, <=, >=
  • Logical operators - e.g. &&, ||, !

Here's an example:

var x = 5;
var y = 10;
var z = x + y;
console.log(z); // Output: 15

In this example, we've declared three variables - x, y, and z - and used the arithmetic operator + to add x and y and assign the result to z.

Conditional Statements

Conditional statements allow us to execute different code based on different conditions. In JavaScript, we use the if, else if, and else keywords to create conditional statements. Here's an example:

If Statement

The if statement is used to execute a block of code if a condition is true. Here's an example:

var x = 10;
if (x > 5) {
  console.log("x is greater than 5");
}

In this example, the if statement checks if the variable x is greater than 5. Since x is equal to 10, the condition is true and the code inside the curly braces will be executed, resulting in the output "x is greater than 5".

Else Statement

The else statement is used to execute a block of code if the condition in the if statement is false. Here's an example:

var x = 3;
if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}

In this example, the if statement checks if the variable x is greater than 5. Since x is less than 5, the condition is false and the code inside the curly braces for the else statement will be executed, resulting in the output "x is less than or equal to 5".

Else If Statement

The else if statement is used to test multiple conditions. Here's an example:

var x = 3;
if (x > 5) {
  console.log("x is greater than 5");
} else if (x < 5) {
  console.log("x is less than 5");
} else {
  console.log("x is equal to 5");
}

In this example, the if statement checks if the variable x is greater than 5. Since x is less than 5, the first condition is false, and the else if statement checks if x is less than 5. Since x is less than 5, this condition is true and the code inside the curly braces for the else if statement will be executed, resulting in the output "x is less than 5".

Ternary Operator

The ternary operator provides a shorthand way to write if-else statements. Here's an example:

var x = 10;
var message = (x > 5) ? "x is greater than 5" : "x is less than or equal to 5";
console.log(message);

In this example, the ternary operator checks if the variable x is greater than 5. Since x is equal to 10, the condition is true and the value of message will be "x is greater than 5".

Switch Statement

The switch statement is used to execute different actions based on different conditions. Here's an example:

var day = "Monday";
switch (day) {
    case "Monday":
      console.log("Today is Monday");
      break;
    case "Tuesday":
      console.log("Today is Tuesday");
      break;
    default:
      console.log("Today is not Monday or Tuesday");
}

In this example, the switch statement checks the value of the variable day and executes the corresponding code block. Since day is equal to "Monday", the first case is executed, resulting in the output "Today is Monday".

Loops

Loops are used to execute the same block of code multiple times. In JavaScript, we have several types of loops:

  • For loop
  • While loop
  • Do-while loop
  • For-in loop
  • For-of loop

For loop

The for loop is used to execute a block of code a specified number of times. Here's an example:

for (var i = 0; i < 5; i++) {
  console.log(i);
}

In this example, we've declared a variable i and assigned it the value of 0. We've then used the for loop to execute the block of code inside the curly braces 5 times, incrementing the value of i by 1 each time.

While loop

The while loop is used to execute a block of code while a specified condition is true. Here's an example:

var i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

In this example, we've declared a variable i and assigned it the value of 0. We've then used the while loop to execute the block ofcode inside the curly braces while the condition i < 5 is true. The value of i is incremented by 1 inside the loop to avoid an infinite loop.

Do-while loop

The do-while loop is similar to the while loop, but it will execute the block of code at least once, even if the condition is false. Here's an example:

var i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

In this example, we've declared a variable i and assigned it the value of 0. We've then used the do-while loop to execute the block of code inside the curly braces at least once, and continue to execute it while the condition i < 5 is true. The value of i is incremented by 1 inside the loop to avoid an infinite loop.

Arrays

Arrays are used to store multiple values in a single variable. In JavaScript, we can declare arrays using square brackets. Here's an example:

var fruits = ['apple', 'banana', 'cherry'];

In this example, we've declared an array called fruits and assigned it the values of 'apple', 'banana', and 'cherry'.

We can access the values of an array using square bracket notation and the index of the value we want to access. Here's an example:

console.log(fruits[0]); // Output: 'apple'

For-of loop

The for-of loop is used to loop through the elements of an iterable object, such as an array. Here's an example:

var fruits = ['apple', 'banana', 'cherry'];
for (var fruit of fruits) {
  console.log(fruit);
}

In this example, we've declared an array called fruits with values 'apple', 'banana', and 'cherry'. We've then used the for-of loop to loop through the elements of the array and log each value to the console.

Functions

A function is a set of instructions that performs a specific task. Here's an example of how to create a function:

function addNumbers(num1, num2) {
  return num1 + num2;
}

In this example, we've created a function called addNumbers, which takes in two parameters num1 and num2, and returns their sum.

You can call a function by using its name followed by parentheses, passing in any required arguments. Here are some examples:

var result = addNumbers(5, 10);
console.log(result); // Output: 15

In this example, we're calling the addNumbers function and passing in the arguments 5 and 10. The function returns their sum, which is assigned to the variable result. The console.log() statement then outputs the value of result, which is 15.

Functions can also be assigned to variables, which is known as a function expression. Here's an example:

var multiplyNumbers = function(num1, num2) {
  return num1 * num2;
};

In this example, we've created a function expression and assigned it to the variable multiplyNumbers. The function takes in two parameters num1 and num2, and returns their product.

You can then call the function by using the variable name followed by parentheses, passing in any required arguments. Here's an example:

var result = multiplyNumbers(3, 4);
console.log(result); // Output: 12

In this example, we're calling the multiplyNumbers function and passing in the arguments 3 and 4. The function returns their product, which is assigned to the variable result. The console.log() statement then outputs the value of result, which is 12.

Objects

An object in JavaScript is a collection of properties, which are key-value pairs. Here's an example of how to create an object:

var person = {
firstName: "John",
lastName: "Doe",
age: 30
};

In this example, we've created an object called person, which has three properties: firstName, lastName, and age, with corresponding values "John", "Doe", and 30.

You can access an object's properties using dot notation or bracket notation. Here are some examples:

console.log(person.firstName); // Output: "John"
console.log(person["lastName"]); // Output: "Doe"

In these examples, we're accessing the firstName and lastName properties of the person object using dot notation and bracket notation, respectively.

You can add or modify properties of an object by simply assigning a value to a new or existing property. Here are some examples:

person.age = 40;
person.job = "Developer";

In these examples, we're modifying the age property of the person object to be 40, and adding a new property called job with a value of "Developer".

Creating an Object

An object in JavaScript is a collection of properties, which are key-value pairs. Here's an example of how to create an object:

var person = {
  firstName: "John",
  lastName: "Doe",
  age: 30
};

In this example, we've created an object called person, which has three properties: firstName, lastName, and age, with corresponding values "John", "Doe", and 30.

Accessing Object Properties

You can access an object's properties using dot notation or bracket notation. Here are some examples:

console.log(person.firstName); // Output: "John"
console.log(person["lastName"]); // Output: "Doe"

In these examples, we're accessing the firstName and lastName properties of the person object using dot notation and bracket notation, respectively.

Adding and Modifying Object Properties

You can add or modify properties of an object by simply assigning a value to a new or existing property. Here are some examples:

person.age = 40;
person.job = "Developer";

In these examples, we're modifying the age property of the person object to be 40, and adding a new property called job with a value of "Developer".

For-in loop

The for-in loop is used to loop through the properties of an object. Here's an example:

var person = {name: 'John', age: 30, city: 'New York'};
for (var key in person) {
  console.log(key + ': ' + person[key]);
}

In this example, we've declared an object called person with properties name, age, and city. We've then used the for-in loop to loop through the properties of the object and log the property name and value to the console.

Object Methods

An object method is a property that contains a function definition. Here's an example:

var person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

In this example, we've added a new property to the person object called fullName, which is a function that returns the person's full name by concatenating the firstName and lastName console.log(person.fullName()); // Output: "John Doe"

In this example, we're calling the fullName method of the person object using dot notation, which returns the person's full name.

Expected Results

If you run the above code, you should see the following output:

John
Doe
40
Developer
John Doe

The first four lines of output correspond to the console logs in the "Accessing Object Properties" and "Adding and Modifying Object Properties" sections. The last line of output corresponds to the console log in the "Object Methods" section.
