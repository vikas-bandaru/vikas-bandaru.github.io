Horizontal Rule (hr) tag

hr is an HTML element used to insert a horizontal rule, which is a visual divider between sections of content. It is a self-closing tag that does not require a closing tag. The hr element creates a line across the entire width of the containing element by default. It is often used to separate different sections of a web page, such as paragraphs, images, or other content.


        <hr>
        <div>
            <hr>
        </div>
        <br>
        <div style="width: 50%;">
            <hr>
        </div>


Div (div) tag

div element is a block-level container element. This means that it takes up the full width of its parent element and creates a new line before and after its contents. It is typically used to group larger sections of content together, such as entire sections of a web page, or to apply styles to a group of elements.


        <div>
            <p>This is a paragraph inside a div tag.</p>
        </div>

This is a paragraph inside a div tag.

What tag is used to create a container for grouping HTML elements?

Span (span) tag

span element is an inline container element. This means that it only takes up as much width as necessary to contain its contents, and does not create new lines before or after its contents. It is typically used to group smaller pieces of content together within a larger block-level element, or to apply styles to a specific portion of text within a block of text.


        <p>This is some <span style="color: red;">text</span> inside a paragraph tag.</p>
