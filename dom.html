JavaScript DOM Manipulation

In this page, we'll learn how to manipulate the DOM using plain JavaScript.

General steps for doing DOM manipulation:

  1. Select the element(s) you want to manipulate using one of the selection methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll.
  2. Use a method or property to manipulate the selected element(s). Common methods and properties include innerHTML, textContent, setAttribute, style, and classList.
  3. Use event listeners to detect when the user interacts with the page, such as clicking a button or scrolling the mouse. Common event listeners include click, submit, keydown, and scroll.
  4. Write JavaScript code to handle the events and perform the desired actions. This can include changing the content of the page, adding or removing elements, or updating the style.

Example 1: Changing Text Content

Click the button to change the text content of the paragraph:

This is a sample paragraph.

<button onclick="changeText()">Change Text</button>
<p id="example1">This is some text.</p>
<script>
function changeText() {
  var para = document.getElementById("example1");
  para.innerText = 'The text content has been changed.';
}   </script>

Example 2: Changing Styling

Click the button to change the background color of the paragraph:

This is another sample paragraph.

<button onclick="changeStyle()">Change Style</button>
<p id="example2">This is another sample paragraph.</p>
<script>
function changeStyle() {
  var para = document.getElementById("example2");
  para.style.backgroundColor = "yellow";
}   </script>

Try it Yourself

Apply red solid border around the paragraph.

  • element.style.property: sets or gets the value of a CSS property for an element
  • element.classList.add(): adds a class name to an element
  • element.classList.remove(): removes a class name from an element
  • element.classList.toggle(): toggles a class name on an element. toggle() is a method in JavaScript that can be used to toggle between adding and removing a specified class from an element.
  • window.getComputedStyle(): returns the computed style of an element

    Computed style is a property of a DOM element in JavaScript that represents the final computed values of all CSS properties applied to the element, after taking into account any styles inherited from parent elements and any styles applied by CSS rules. It is accessed using the window.getComputedStyle() method.

let myDiv = document.getElementById('my-div');
myDiv.style.backgroundColor = 'blue';
myDiv.classList.add('my-class');
myDiv.classList.remove('my-class');
myDiv.classList.toggle('my-class');
let myStyle = window.getComputedStyle(myDiv);
console.log(myStyle.backgroundColor);

Example 3: Creating Elements

Click the button to create a new paragraph:

<button onclick="createParagraph()">Click me</button>
<script>
function createParagraph() {
  var p = document.createElement("p");
  var text = document.createTextNode("This is a newly created paragraph.");
  var button = document.getElementsByClassName("exbtn")[2];
  button.parentNode.insertBefore(p, button);  }  </script>

Try it Yourself

Create a DIV and add a child Paragraph under it.

  • document.createElement(): creates a new HTML element
  • element.appendChild(): adds a child element to another element
  • element.innerHTML: sets or gets the inner HTML of an element
  • element.setAttribute(): sets the value of an attribute for an element
  • element.style.property: sets or gets the value of a CSS property for an element
  • parent.insertBefore(p, button); The parent is a variable that represents the parent element of button. The method insertBefore() takes two arguments: the first argument p is the element to be inserted, and the second argument button is the element before which p will be inserted.
let myDiv = document.createElement('div');
myDiv.innerHTML = '<strong>Hello World!</strong>';
myDiv.setAttribute('id', 'my-div');
myDiv.style.color = 'red';
document.body.appendChild(myDiv);
parent.insertBefore(p, button);

Example 4: Looping through Elements

Click the button to change the text of all paragraphs:

This is the first paragraph.

This is the second paragraph.

This is the third paragraph.

<button onclick="changeAllParagraphs()">Change Text</button>
<script>
function changeAllParagraphs() {
  var paragraphs = document.getElementsByClassName("para");
  for (var i = 0; i < paragraphs.length; i++) {
    paragraphs[i].innerText = "This text has been changed.";
  }
}  </script>
